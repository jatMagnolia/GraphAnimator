# GraphAnimator

A drawing program designed to help visualize data structures in computer science. Create interactive diagrams with boxes, circles, text, and connections to illustrate how data structures work and interact.

## Features

- **Drawing Tools**: Create boxes, circles, and text objects on a canvas
- **Connections**: Draw connections between objects with directional arrows
- **Grouping**: Split and group boxes together with shared connections
- **Select & Lasso**: Select objects individually or with a lasso tool
- **Resize**: Resize objects with interactive handles
- **Grid Background**: Visual grid for precise alignment
- **Color Customization**: Change object colors with a color picker

## Current Status

This project is in active development. The current version provides basic drawing and connection functionality.

## Future Plans

- **Animation**: Add animation capabilities to visualize data structure operations
- **Download**: Export diagrams as images or other formats
- **Interactive Visualizations**: Create animated demonstrations of data structure algorithms

## Getting Started

Simply open `index.html` in a web browser to start using the drawing program.
